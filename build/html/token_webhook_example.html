token_webhook_example module¶

class token_webhook_example.Discord¶

Bases: object

A class for interacting with Discord webhooks.

This class provides methods to send messages to a Discord channel using webhooks. It is initialized with a webhook URL and offers functionality to format and send messages in a streamlined manner.

webhook_url¶

The URL of the Discord webhook used to send messages.

Type:

str

send_webhook(message_description: str) → None¶

Sends a message to the Discord channel associated with the webhook.

Parameters:

message_description (str) – The message to be sent to the Discord channel.
